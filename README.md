# Background Remover Web App

This web application allows users to upload an image, preview it, and remove its background with a single click. The cleaned image is displayed instantly and can be downloaded using a custom filename. The site includes a smooth dark theme, animated mouse-move background effects, and a user-friendly layout.

Built using **HTML, CSS, JavaScript**, and a **Python Flask backend** that handles image processing via a `/remove-bg` endpoint.

## Features

- Upload & preview images  
- Auto background removal  
- Live filename input for download  
- Animated UI with light trail effect  
- Python backend for background removal

> **Made for demo and learning purposes.**
